# etchosts.link

Preview a website before DNS propagation without editing your hosts file.

When migrating a website to a new server you often need to test the site before switching DNS records. Normally this requires editing the hosts file locally, which only works on your own machine.

etchosts.link generates a preview link that routes the domain to a specific server IP while preserving the Host header.

This allows you to test a website before DNS propagation without changing system configuration.

## Use cases

• Website migrations  
• Testing staging servers  
• Previewing deployments before DNS changes  
• Sharing preview links with teammates or clients
